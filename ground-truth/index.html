What is Ground Truth MCP?

Ground Truth is a Model Context Protocol (MCP) server that provides AI agents with fact-checking and market research tools. Instead of hallucinating or relying on stale training data, agents can check if an API endpoint actually exists, count real market competitors on npm or PyPI, extract current pricing from websites, and cross-reference claims against live sources. All results come from data fetched in real-time, with 5-minute caching for performance.

AI Agents Guess. Ground Truth Checks.

When you ask an AI agent "does this API exist?" or "what does Stripe charge?", it answers from training data that might be months or years old. Ground Truth gives agents the ability to check the actual source in real-time.

Frequently Asked Questions

What is Ground Truth MCP? +
Ground Truth is a Model Context Protocol (MCP) server that gives AI agents the ability to validate claims, check API endpoints, estimate market size, extract pricing data, compare packages, and test hypotheses — all against real-time live web data instead of training data.
What is an MCP server? +
MCP (Model Context Protocol) is an open standard created by Anthropic for connecting AI models to external tools and data sources. An MCP server exposes tools that AI agents can call. Ground Truth exposes fact-checking and market research tools.
Which AI agents work with Ground Truth? +
Any MCP-compatible agent. This includes Claude (via Claude Desktop or Claude Code), ChatGPT with MCP plugins, Cursor, Windsurf, Continue, and any custom agent built with the MCP SDK.
Can I self-host it? +
Yes. Ground Truth is open source on GitHub. Clone the repo, deploy to your own Cloudflare Workers account, and you're set. The hosted version is also available for instant use.
How does caching work? +
Results are cached for 5 minutes using Cloudflare Durable Objects with SQLite. This means repeated queries for the same URL or package return instantly while staying reasonably fresh. After 5 minutes, the next request fetches live data again.
Is there a usage limit? +
No hard usage limits on either tier. The free tier includes unlimited check_endpoint calls. The Pro tier at $9/month includes unlimited usage of all 6 tools. Fair use applies.
